Bundled US OpenCelliD CSV data

The app uses these files as the initial tower data source for the map (USA MCC 310, 311, 312, 313, 314).
Towers are loaded from here first—no API key or download needed. Downloaded/cached files are used only if a bundled file is missing.

Expected filenames: 310.csv.gz, 311.csv.gz, 312.csv.gz, 313.csv.gz, 314.csv.gz
(or plain .csv). Source: OpenCelliD (opencellid.org) MCC export.
